Requirements:
	Godot v4.3.dev2.official [352434668]
	Ein Server auf dem die KI läuft (Server Script verwenden)

Anleitung:

1. Im Addon Script "logic.gd" die IP durch die des Servers austauschen und speichern
2. Ordner in den Addon Ordner in der Engine einfügen
3. Oben mittig auf "AssetLib" klicken
4. Rechts oben auf "Plugins..." klicken
5. Haken bei "Aktivieren" setzen
